1. Tujuan

  • Peserta didik dapat memahami dasar pemrograman server-side menggunakan HTML.
  • Peserta didik menguasai sintaks-sintaks dasar HTML.
  • Peserta didik mampu menyelesaikan permasalahan sederhana menggunakan HTML.

2. Dasar Teori

HTML (HyperText Markup Language) merupakan bahasa markup untuk menyusun halaman web yang dapat dibaca oleh browser.

Browser seperti Google Chrome atau Firefox akan menerjemahkan kode HTML menjadi tampilan visual halaman web.

Beberapa Elemen HTML yang Sering Digunakan:
Tag Fungsi
<h1>-<h6>Judul atau heading
<p>Paragraf teks
<a>Hyperlink
<img>Menampilkan gambar
<br>Ganti baris
<hr>Garis pemisah horizontal
<b>Teks tebal
<i>Teks miring

3. Langkah Percobaan

  1. Buka aplikasi Visual Studio Code.
  2. Buat file baru dengan nama index.html.
  3. Ketikkan struktur dasar HTML berikut:
<!DOCTYPE html>
<html>
<head>
  <title>Halaman Pertama Saya</title>
</head>
<body>
  <h1>Selamat Datang</h1>
  <p>Ini adalah halaman web pertama saya.</p>
</body>
</html>
  1. Simpan file tersebut dan buka menggunakan browser.
  2. Perhatikan hasilnya dan identifikasi bagian-bagian dari struktur HTML.

4. Tugas

  1. Buatlah halaman HTML yang berisi biodata singkatmu.
  2. Gunakan elemen seperti <h1>, <p>, <img>, dan <a>.
  3. Tampilkan hasilnya pada browser.

5. Penutup

Pada praktikum ini peserta didik telah mempelajari dasar-dasar HTML untuk membuat struktur halaman web sederhana. Penguasaan HTML menjadi langkah awal sebelum melanjutkan ke pemrograman web yang lebih kompleks seperti CSS dan JavaScript.
